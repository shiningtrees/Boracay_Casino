## 주요 발견

### 1. 백테스트 엔진 검증 완료 ✅

**정상 작동 확인:**
- MEXC API 데이터 다운로드
- OHLCV 5분봉 처리
- High/Low 기준 손절/익절 체크
- 거래 비용 0.3% 반영
- 다중 주기 비교
- JSON 결과 저장

**실행 성능:**
- 2000 캔들 로드: ~2초
- 주기별 시뮬레이션: ~1초
- 총 실행 시간: 3주기 약 6초

### 2. 청산 사유 분석

**이번 테스트에서:**
- **timeout만 발생**: 모든 거래가 타임아웃으로 청산
- **손절/익절 미발동**: 7일 기간 동안 -25% 손절이나 +25% 트레일링 조건 미도달

**이유:**
- BTC/ETH는 변동성이 낮음 (메이저 코인)
- 7일 기간이 짧음
- 실전에서는 알트코인(잡코인)을 대상으로 함

### 3. 데이터 제약

**MEXC API 제한:**
- limit=500 최대 (요청당)
- 2024년 과거 데이터 조회 불가
- 최근 7~10일 데이터만 가능

**해결 방안:**
- 더 긴 기간 백테스트를 위해서는 데이터 수집 스크립트 별도 필요
- 또는 다른 데이터 소스 활용 (Binance, CoinGecko 등)
